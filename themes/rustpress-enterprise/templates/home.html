  • Memory Safe

    No buffer overflows, null pointer dereferences, or data races

  • Blazing Fast

    Compiled to native code, no runtime overhead

  • Concurrent

    Async/await with Tokio for maximum throughput

  • Type Safe

    Catch bugs at compile time, not in production

Get Started in Minutes

Terminal 
# Clone and run RustPress
$ git clone https://github.com/rustpress/rustpress
$ cd rustpress
$ cargo run

# Open in browser
Server running at http://localhost:8080
Admin panel at http://localhost:8080/admin

# Start creating content!
