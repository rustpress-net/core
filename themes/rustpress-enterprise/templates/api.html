Quick Start

Get started in minutes

1

Get your API key

Generate an API key from your account settings. Keep it secure!

Dashboard → Settings → API Keys 
API_KEY=sk_live_xxxxxxxxxxxx
2

Install the SDK

Choose your preferred language and install the official SDK.

npm install @rustpress/sdk
pip install rustpress
gem install rustpress
go get github.com/rustpress/rustpress-go
3

Make your first request

Initialize the client and start making API calls.

example.js 
import { RustPress } from '@rustpress/sdk';

const client = new RustPress({
  apiKey: process.env.RUSTPRESS_API_KEY
});

// Get all pages
const pages = await client.pages.list();
console.log(pages);

// Create a new post with AI enhancement
const post = await client.posts.create({
  title: 'My New Post',
  content: 'Post content here...',
  ai_enhance: true
});
console.log(post);

API Reference

Authentication

3 endpoints
POST /v1/auth/token Generate access token
POST /v1/auth/refresh Refresh access token
DELETE /v1/auth/revoke Revoke token

Users

5 endpoints
GET /v1/users List all users
GET /v1/users/:id Get user by ID
POST /v1/users Create user
PATCH /v1/users/:id Update user
DELETE /v1/users/:id Delete user

Records

6 endpoints
GET /v1/records List records
GET /v1/records/:id Get record

PATCH /v1/records/:id Update record
DELETE /v1/records/:id Delete record
POST /v1/records/bulk Bulk operations
